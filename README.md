# 用電大戶法規說明簡報

供業務／顧問對業主(客戶)口頭說明時使用的 HTML 互動簡報。內容涵蓋:

- **現行義務**:《再生能源發展條例》第 12 條(5,000kW / 10% / 四選一)
- **修法動向**:《能源管理法》修正草案(發電＋儲能強制、稽查與罰則)
- **現行 vs. 草案 對比表**、對貴公司的影響情境、建議下一步

## 使用方式

- 直接開啟 `index.html`,或透過 GitHub Pages 網址分享給業主
- **鍵盤**:`← →` 或空白鍵切頁、`F` 全螢幕、`Home / End` 跳首末頁
- **滑鼠**:點畫面左／右側切頁
- **手機**:左右滑動切頁(RWD 支援)

## 特色

- 單一 HTML 檔,無需安裝、離線可用、一鍵分享連結
- 專業能源藍主題,適合對外客戶簡報
- 10 頁 · 約 5 分鐘說明時長

## 免責聲明

本簡報所引述之《能源管理法》修正草案為立法委員提案版本,尚未三讀,內容可能修正或不通過,僅供超前部署參考,不構成法律義務。實際義務內容以最終公布之法規及子法為準。
