# 用電大戶法規與表後儲能效益｜顧問簡報

供業務／顧問對業主與客戶說明用的 HTML 互動簡報,採 **BCG 顧問簡報格式**(SIP 原則:簡潔 Simple、洞察 Insightful、金字塔 Pyramidal)。一份簡報依序涵蓋兩大主題:

- **Part 1｜用電大戶法規遵循** — 現行 10% 綠能義務、《能源管理法》修法動向(發電＋儲能強制)、對客戶的影響與行動
- **Part 2｜表後儲能應用效益** — 三層效益疊加:降容防超約止血 → 套利／需量反應／電力交易(VPP)／大戶義務回饋 → 微電網與淨尖峰加值

## 格式特色(BCG SIP)

- **標題即結論**:每頁標題是「主要訊息／論點(洞察)」,而非描述性題目
- **金字塔結構**:標題(論點)在上、下方為佐證圖表/表格
- **頁首標籤**:左標籤=主題章節,右標籤=資料狀態(現行有效／審議中／初步試算須核對電價)
- **來源註腳 + Executive Summary**:一頁看完的結論摘要放在最前
- 白底 + BCG 綠(#177B57)專業配色,單一 HTML、離線可用、一鍵分享

## 使用方式

- 開啟 `index.html`,或透過 GitHub Pages 網址分享
- **鍵盤**:`← →`／空白鍵切頁、`F` 全螢幕、`Home / End` 跳首末頁
- **滑鼠**:點畫面左／右側切頁 · **手機**:左右滑動(RWD)
