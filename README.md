# 用電大戶法規與表後儲能效益｜顧問簡報

供業務／顧問對業主與客戶說明用的 HTML 互動簡報,採**編輯型「討論用工作稿」風格**(teal 綠 + 珊瑚紅 + 琥珀色、暖色紙感底、大標題、軟色調數據卡、深色重點區塊),並依《AI 簡報總導演》方法論製作(標題即結論、每頁一重點、減法美學、圖表灰階+單一強調色)。一份簡報涵蓋兩大主題:

- **Part 1｜用電大戶法規遵循** — 現行 10% 綠能義務、《能源管理法》修法動向(發電＋儲能強制)、對客戶的影響與行動
- **Part 2｜表後儲能應用效益** — 三層效益疊加:降容防超約止血 → 套利／需量反應／電力交易(VPP)／大戶義務回饋 → 微電網與淨尖峰加值

## 格式特色

- **標題即結論**:每頁大標是「主要訊息／論點」,下方為佐證表格圖表
- **頁首 eyebrow + 右上狀態旗標**:現行有效(綠)／審議中(琥珀)／初步試算·須核對電價(珊瑚)
- **元件**:軟色調 metric 卡、深色 big-callout 重點塊、teal 圓點清單、時間軸步驟、來源註腳
- 白紙感底 `#f6f8f7` + teal 綠 `#0f8f70`,單一 HTML、離線可用、一鍵分享

## 使用方式

- 開啟 `index.html`,或透過 GitHub Pages 網址分享
- **首頁 Hub**:兩張 Part 卡片＋完整目錄,點任一頁直接跳入;頂部導覽列隨時切換 首頁／Part 1／Part 2
- **鍵盤**:`← →`／空白鍵切頁(Part 1 末頁按 → 自動接 Part 2)、`F` 全螢幕、`Home` 回首頁、`End` 跳末頁
- **按鈕**:右下角 `‹ ›` 上下頁＋頁碼 · 底部三色進度條 · **手機**:左右滑動(RWD,980px 以下單欄堆疊)
- **網址可分享到特定頁**:如 `#p2-3`(Part 2 第 3 頁)
